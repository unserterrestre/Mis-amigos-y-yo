# Mis-amigos-y-yo
Selamat datang di program kami!

Dengan bersenang hati, kami mempersembahkan program yang dibuat oleh kami secara bersama-sama kepada khalayak umum. Program ini bertujuan melangsungkan perhitungan matematis yang sederhana, yakni pertambahan, pengurangan, perkalian, dan pembagian. Struktur sintaktik tidak terlalu rumit; kami menerapkan perulangan _while_ dan logika selektif _if_, serta memberikan arahan kepada pengguna dan menerima masukan pengguna dengan memakai fungsi masukan dan keluaran. Pembuatannya diharapkan dapat memenuhi nilai salah satu tugas dalam mata kuliah Pemrograman Dasar dan Praktikum.
